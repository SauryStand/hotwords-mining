话题检测

Topic detection

使用twitter Stream API 提供的实时数据。 可以追踪特定关键词的推文、某些用户的推文、在某些特定的地理位置（GEO）的推文。
使用变分推断的动态词库的WOLDA算法，挖掘实时主题中的关键词，并跟踪话题趋势。

情感分析

Sentiment analysis

使用twitter search API提供的数据进行查询。
使用最大熵分类器,训练数据来自 SemEval比赛
可以帮助:

  • 用户查询人们谈论某些特定词时候，带有的感情色彩
  • 商家查看消费者对于新推出的产品的评价
